# 🌐 AI 双语助手 v1.0.0

**让中国用户与 AI 平台无障碍交流的 Chrome 扩展**

一键翻译，智能保留代码格式，支持 ChatGPT、Gemini、Claude。

---

## ✨ 功能特点

### 🎯 输入翻译
- 自动检测中文输入
- 一键翻译成英文
- 智能跳过代码块

### 📖 选中翻译
- 选中任意英文文字
- 点击按钮翻译成中文
- **代码和公式完全保留原样**

### 🎨 现代UI
- 紧凑悬浮窗口
- 可拖动、可缩放
- 自动适配深色模式

## 📦 安装方法（小白教程）

### 第一步：下载插件

1. 点击绿色按钮 **"Code"** → **"Download ZIP"**
2. 解压下载的文件到任意位置（记住这个位置）

### 第二步：打开 Chrome 扩展管理

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions`
3. 按回车键

### 第三步：开启开发者模式

1. 在页面右上角找到 **"开发者模式"** 开关
2. 点击打开它（开关变蓝色）

### 第四步：加载插件

1. 点击左上角 **"加载已解压的扩展程序"** 按钮
2. 选择刚才解压的文件夹
3. 点击 **"选择文件夹"**

### 第五步：完成！

- 你会看到插件图标出现在浏览器右上角
- 访问 [ChatGPT](https://chat.openai.com)、[Gemini](https://gemini.google.com) 或 [Claude](https://claude.ai) 开始使用

## 🚀 使用方法

### 输入翻译

1. 打开 ChatGPT / Gemini / Claude
2. 在输入框输入中文
3. 右下角悬浮窗显示翻译预览
4. 点击 **"✓ 使用"** 替换输入框内容

### 选中翻译

1. 选中 AI 回答中的英文内容
2. 点击出现的 🔄 按钮
3. 文字被翻译成中文（代码保持原样）
4. 点击翻译后的文字可切换回原文

### 窗口操作

| 操作 | 方法 |
|------|------|
| 打开窗口 | 点击右下角 🌐 按钮 |
| 关闭窗口 | 点击 − 按钮 |
| 移动窗口 | 在窗口任意位置拖动 |
| 调整大小 | 拖动右下角 |

## 💡 常见问题

### Q: 翻译按钮不出现？
A: 选中的文字需要至少5个字符。

### Q: 代码被翻译了？
A: 正常情况下代码会自动保留。如果仍然出现问题，请刷新页面重试。

### Q: 支持哪些网站？
A: 目前支持 ChatGPT、Gemini、Claude。其他网站可以使用选中翻译功能。

### Q: 翻译失败？
A: 请检查网络连接，本插件使用 Google 翻译服务。
