# 🎯 三分钟安装指南

## 步骤一：下载

点击页面上的绿色按钮 **Code** → **Download ZIP**

或者直接下载这个文件夹。

---

## 步骤二：打开 Chrome 扩展页面

在 Chrome 浏览器地址栏输入：

```
chrome://extensions
```

然后按回车键。

---

## 步骤三：开启开发者模式

在页面 **右上角** 找到 "开发者模式" 开关，**点击打开**。

```
┌─────────────────────────────────────────────┐
│ 扩展程序                    [开发者模式 🔘] │
└─────────────────────────────────────────────┘
                                    ↑
                               点这里打开
```

---

## 步骤四：加载插件

点击左上角 **"加载已解压的扩展程序"** 按钮。

```
┌─────────────────────────────────────────────┐
│ [加载已解压的扩展程序]  [打包扩展程序]      │
│      ↑                                      │
│   点这里                                    │
└─────────────────────────────────────────────┘
```

然后选择你解压的 **ai-bilingual-assistant** 文件夹。

---

## 步骤五：完成！

你会看到插件出现在列表中：

```
┌─────────────────────────────────────────────┐
│ 🌐 AI 双语助手                              │
│    版本 1.0.0                               │
│    [详情]  [移除]                   [启用 ✓]│
└─────────────────────────────────────────────┘
```

---

## 开始使用

1. 访问 https://gemini.google.com
2. 在输入框输入中文
3. 查看右下角悬浮窗口的翻译预览
4. 点击 **"✓ 使用"** 替换输入

---

## 遇到问题？

### 看不到插件图标？
点击 Chrome 右上角的拼图图标 🧩，把 "AI 双语助手" 固定到工具栏。

### 翻译不工作？
刷新页面，或者重新加载扩展（在 chrome://extensions 点击刷新按钮）。

### 其他问题？
请在 GitHub 提交 Issue 或联系开发者。
